# E-todo Project

## The purpose of the project

E-todo is an application that allows managing tasks for different users.

The project contains:

- a MySQL database
- a Node.js / Express API
- a front-end with React

## Required installations

Before starting the project, make sure you have installed:

- Node.js
- Docker & Docker Compose

## Commands to start the server

### With Docker

At the root:

`docker-compose up --build`

In the Frontend:

`npm install`
`npm start`

### With Node (without Docker)

In the backend folder AND in the frontend folder:

`npm install`  
`npm start`
